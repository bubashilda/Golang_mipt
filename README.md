## Программа курса

- [helloworld](./helloworld) - знакомство с базовым синтаксисом Go: функции, пакеты, работа со строками, HTTP-запросы
- [basics](./basics) - углубленное изучение базовых конструкций: структуры, методы, интерфейсы, работа с памятью
- [interfaces](./interfaces) - освоение интерфейсов, полиморфизма и паттернов проектирования в Go
- [goroutines](./goroutines) - работа с горутинами, примитивами синхронизации и конкурентным программированием
- [testing](./testing) - написание тестов, использование testing пакета, проверка утечек ресурсов
- [gitfame](./gitfame) - разработка CLI-приложений, работа с файловой системой и внешними командами
- [shmem](./shmem) - паттерны конкурентного программирования: пулы, батчинг, публикация-подписка
- [http](./http) - создание HTTP-серверов, middleware, REST API и работа с сетью
- [generics](./generics) - использование обобщенного программирования в Go (добавлено в Go 1.18+)
- [reflect](./reflect) - работа с reflection API для динамического анализа и манипуляции типами
